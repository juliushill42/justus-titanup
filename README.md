# JustUs: Autonomous Legal Sovereign Node (v3.0)

An enterprise-grade legal orchestration platform engineered for high-density multi-device field execution, automated document compilation, and immutable evidence authentication.

## ↗ System Architecture

- **Enterprise Cloud Intelligence (Vertex AI Maas):** Connects to Google Cloud's Vertex AI (`meta/llama3-405b-instruct-maas`) for cost-effective, scalable processing with zero local hardware load.
- *'Zero-Knowledge Privacy Vault:** Computes SHA-256 signatures locally in-browser via the Web Crypto API. Raw files never hit central servers unencrypted, eliminating data-hosting liabilities.
- **Institutional Admissibility Moat:** Automatically generates machine-readable JSON chain-of-custody ledgers satisfying Federal Rules of Evidence Rule 901(b)(9) and Rule 902(13).
- **Adaptive Fluid Breakpoints:** Fully responsive Tailwind CSS layout auto-stacks on mobile screen frames and expands seamlessly into administrative layouts on desktop displays.
- **Header-Intercepting Global Localization (i18n):** Handles dynamic multilingual directory switching (`en`/`es`) by reading incoming browser Accept-Language headers.
